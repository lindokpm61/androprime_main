What changed from v1

  • Killed the lab-form table. It read as a medical procedure, not a premium product. Replaced with space + a monogram seal + a serif product name.
  • Rigour is now a whisper, not a shout: a small reference code (REF · AP–T01), one understated marker line, and the IVD / CE / UKAS marks shrunk to quiet fine print at the foot.
  • QR removed from the front — it moves to the back/inside, where setup instructions live. The front sells; the back instructs.
  • Serif product name (Merriweather) for a heritage / premium note against the mono detailing. Still within brand type.
  • Still nothing scary: no "clinical examination" language. "At-home hormone test" reads premium-consumer, not medical.
  • Bracketed / placeholder specifics (marker count, turnaround) live on the side panel, not the face. The face stays clean.
