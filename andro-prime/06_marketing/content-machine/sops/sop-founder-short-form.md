# SOP: Founder Short-Form

**Idea to scheduled short (Reel / Short / TikTok), in Keith's voice, compliance-cleared.** Read `founder-content-system.md` first.

**Trigger:** a slot on the calendar, or a genuine founder moment (a retest lands, a result read-through).

**Roles:** agent drafts the script and runs pre-flight; Keith records and edits; Ewa only for a net-new claim.

---

## Steps

1. **Pick the idea** from the yap map / hook bank (`templates/hook-bank.md`) or the founder-journey arc. Confirm it maps to a live-kit marker and to a signed canonical asset (or is Keith's own data, which still cannot assert a supplement caused a change). Running `/hook` and picking a hook mints the asset file (`content-machine/assets/`); everything below happens inside it.
2. **Write the script** to `templates/short-form-script.md`: feeling-first hook in the first line, the four-move Keith arc (moment → question → reframe → question to the viewer), 100-150 words, fragment-dense (`tone-of-voice.md` §6). Fresh wording, not a repeated opener. `/script` writes this straight into the asset file body and advances its status to `scripted`.
3. **Run `/compliance-preflight` on the script** (Guardrail #1). This happens **before recording**, so Keith never films a script that fails. 🔴 = rewrite; 🟠 = Ewa; 🟢 = record. Net-new claim → route to Ewa first.
4. **Keith records.** Phone on a tripod, lav mic, window light. One take where possible. Show the number / range / marker on screen for the muted viewer. If the Daily Stack is on camera, its silent formulation ingredient is never named or shown on the label (silent-ingredient rule, `03_compliance/CONTEXT.md`).
5. **Edit.** Keep the hook in frame one. Add the on-screen text hook and the series tag. Grayscale / brand visual rules for any graphic. Keep it honest over polished.
6. **Thumbnail / cover frame** per `sop-thumbnail.md` (9:16 cover).
7. **Caption + CTA.** Feeling-first caption, UTM bio link to the router, no engagement-bait CTA. Earn the share, do not ask for it.
8. **Final go and schedule.** Keith approves; agent schedules on the calendar. No auto-posting without Keith's go. Record the transition via `/content-status` ("<slug> approved / scheduled / posted <url>"): the gate scanner (`.claude/skills/content-status/scan.js`) blocks scheduling a rendition until its 9x16 thumb is confirmed.

---

## Definition of done

- Script passed pre-flight before recording.
- Hook works muted; series tag present; on-brand visuals.
- No supplement-causation claim, the silent ingredient never named, no FM-list CTA.
- Keith recorded and approved; scheduled.

## Anti-checklist (stop if any is true)

- Recording started before pre-flight passed.
- The script asserts a supplement fixed / cured / caused a change.
- The hook promotes a not-yet-live marker or leans andropause before Ewa clears Pillar E.
