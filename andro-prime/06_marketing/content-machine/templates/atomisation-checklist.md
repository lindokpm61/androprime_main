## Per-derivative checklist

**Derivative:** ______ (YT long-form / Short / Reel / Facebook / LinkedIn / email hook / thumbnail)
**Canonical asset it inherits from:** ______ (the signed pillar)
**Pillar / marker:** ______

- [ ] **Claim ceiling:** makes no claim the canonical asset does not make (if it does, back to canonical + Ewa).
- [ ] **Feeling-first:** opens on the body-feeling; the marker is the answer, not the hook.
- [ ] **Live-kit marker:** the marker maps to a currently-available kit (T; Vit D / B12 / ferritin / hs-CRP). If not, CTA routes to email capture.
- [ ] **Voice:** in Keith's voice; no repeated verbatim opener; no bullet lists on LinkedIn.
- [ ] **Compliance hedge:** signal-not-cause; GP-not-supplement where a result/marker appears.
- [ ] **Hard rails clean:** no diagnose / treat / cure / fix; no "TRT available"; EFSA-approved supplement wording only (the silent ingredient never named); "Ewa-approved recommendation logic" not "reviewed by our doctor"; no em dashes.
- [ ] **CTA routing:** `kitCTA` to Kit 1/2/3 or the email rung. Grep clean for `founding.member`, `FM list`, `join the list`, `priority access to TRT`, `clinical service waitlist`.
- [ ] **Platform shape:** correct format/size/cadence for the channel (`content-machine-blueprint.md` §2); Facebook is a native upload, not an IG cross-post.
- [ ] **Thumbnail:** attached and on-brand (`sops/sop-thumbnail.md`) where the surface needs one.
- [ ] **Pre-flight:** ran `/compliance-preflight`; 🟢 (or 🟠 Ewa-cleared).
- [ ] **Go:** Keith approved; scheduled.

---

**Ship only when every box is checked.** Any unchecked box is a stop.
